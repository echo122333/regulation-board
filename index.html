한눈에 보는 규제혁신·창의제안

창의제안, 규제혁신 우수사례, 법령제도개선 건의과제의 발굴부터 제출·심사·후속조치까지 — 현장의 아이디어가 제도가 되는 과정을 한곳에서 관리합니다.

법령제도개선 건의과제

2026년 지방규제혁신 추진계획 (행안부, '26.2.26.)
개요
지역 현장의 불합리한 중앙법령·제도 규제를 발굴해 서울시(창의규제담당관)를 거쳐 행안부·소관부처에 개선 건의
제출주기
분기별 정기건의 + 수시건의
중점분야
민생규제 4대 분야: 국민생활 불편 해소 / 소상공인·자영업자 영업규제 / 사회적약자 / 생명·안전 (국정19)
지역산업규제 3대 분야: 지역 전통산업 / 서비스산업(콘텐츠·관광·의료 등) / 신산업(AI·드론 등) (국정36)
처리절차
과제 발굴·부처협의 → 현장분석 → 지방규제혁신위원회 검토·조정(민생·지역산업 분과위, 규제책임위원) → 규제합리화위원회 보고
수용률
부처 수용률 약 10% 수준('25년 195건 수용) — 규제의 실효성·적정성 분석과 구체적 대안 제시가 수용률을 좌우
이행점검
수용 과제는 상·하반기 이행여부 점검 — 제출 후에도 수용여부·후속조치 추적 필요
성과평가
행안부 성과평가 4개 지표: 추진계획 수립 / 건의과제 수용실적 / 자치법규 정비실적 / 경진대회 우수사례 선정 → 우수기관 특별교부세·정부포상
