🏢 우리 재단(투자·출연기관) 적용 기준

  • 제출 경로: 자체 제안시스템 등재·심사 후 서울시(창의규제담당관)로 제출 — 재단은 부서별 발굴 → 취합·심사 → 분기별 일괄 제출
  • 포상금: 창의제안상 시상은 동일하나 포상금은 재단 예산 범위 내 지급 (서울시 예산 아님)
  • 우수실국 평가: 市 실·본부·국 대상 제도로 재단은 기관 단위 평가 대상이 아니며, 개인 포상(창의제안상·학습시간) 중심으로 참여
  • 선정 절차·등급 체계(A~D)·평가기준은 서울시 기준과 동일하게 적용

🏢 우리 재단(투자·출연기관) 적용 기준

  • 제출 건수: 투출기관은 1건 이내(규제 총괄부서 수합 후 선정 제출) — 市 본청 부서는 자율, 자치구는 3건 이내
  • 인센티브: 외부 기관이므로 시장상장만 수여 가능, 포상금 지급 제외(공직선거법 제112·113조). 특별휴가는 기관별 자체 부여
  • 심사: 市 본청·사업소·투출기관 그룹으로 함께 심사(1차 2배수 14건 → 2차 1배수 7건 → 본심사)
  • 공적기간·심사기준은 전체 기준과 동일 — 제도개선 '완료' 시점이 공적기간 내여야 함

법령제도개선 건의과제

2026년 지방규제혁신 추진계획 (행안부, '26.2.26.)
개요
지역 현장의 불합리한 중앙법령·제도 규제를 발굴해 서울시(창의규제담당관)를 거쳐 행안부·소관부처에 개선 건의
제출주기
분기별 정기건의 + 수시건의
중점분야
민생규제 4대 분야: 국민생활 불편 해소 / 소상공인·자영업자 영업규제 / 사회적약자 / 생명·안전 (국정19)
지역산업규제 3대 분야: 지역 전통산업 / 서비스산업(콘텐츠·관광·의료 등) / 신산업(AI·드론 등) (국정36)
처리절차
과제 발굴·부처협의 → 현장분석 → 지방규제혁신위원회 검토·조정(민생·지역산업 분과위) → 규제합리화위원회 보고
수용률
부처 수용률 약 10% 수준('25년 195건 수용) — 규제 실효성 분석과 구체적 대안 제시가 관건
이행점검
수용 과제는 상·하반기 이행여부 점검 — 제출 후에도 수용여부·후속조치 추적 필요

🏢 우리 재단(투자·출연기관) 적용 기준

  • 건의기관 구분: 과제 목록 작성 시 '2. 투출기관'으로 분류해 서울시 경유 제출 — 담당자·전문가/부처 의견 청취 여부, 서울시 소관부서 기존 협의여부 기재 필요
  • 재단 특화 발굴 영역: 출자출연기관 자체에 적용되는 규제(예: 「지방출자출연기관 인사·조직 운영지침」의 육아휴직 대체인력 제외 문제)도 건의 대상 — 전국 800여 개 기관에 파급되는 과제로 수용 가능성·효과성이 높음
  • 성과 귀속: 행안부 지방규제혁신 성과평가는 지자체 대상이며 재단 건의 실적은 서울시 실적으로 합산 — 서울시 소관부서와의 사전 협의가 채택률을 높임
